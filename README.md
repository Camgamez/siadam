# Aeronautical and Meteorological Data Integrated System (AMDIS)

An integrated system for consulting airtraffic data and contrast it with meteorological data. 

## Purposes:

The main goal for this project is to assist pilots and crewmembers, despatchers and airport operators alike in schedulling and managing flights based on real-time weather conditions near airports.

## Current features:

- A basic dataentry website where the information can be queried from.
